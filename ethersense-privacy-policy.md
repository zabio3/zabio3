# Privacy Policy for EtherSense

**Last Updated: January 25, 2026**

## Overview

EtherSense ("the App") is a Wi-Fi network diagnostics application. This Privacy Policy explains how the App handles your data.

## Data Collection

### What We Collect

The App accesses the following information on your device:

- **Wi-Fi Network Information**: Signal strength (RSSI), link speed, channel, SSID, and BSSID of connected and nearby networks
- **Location Data**: Required by Android (6.0+) to perform Wi-Fi scans. This is an Android system requirement.
- **Network Configuration**: IP address, gateway, DNS servers for diagnostic purposes
- **Speed Test Results**: Download/upload speeds and latency measurements

### How Data is Processed

**All data is processed locally on your device.** The App does not:

- Send any data to external servers
- Store data in cloud services
- Share data with third parties
- Create user accounts or profiles

The only network connections made by the App are:
- Speed test measurements to standard speed test servers
- DNS lookups for the DNS Lookup tool
- WHOIS queries for the WHOIS tool

## Permissions

The App requires the following permissions:

| Permission | Purpose |
|------------|---------|
| Location (Fine/Coarse) | Required by Android for Wi-Fi scanning |
| Wi-Fi State | Read and modify Wi-Fi connection state |
| Internet | Speed test functionality |
| Vibrate | Haptic feedback |

## Data Storage

All settings and preferences are stored locally on your device using Android's DataStore. No data is transmitted externally.

## Children's Privacy

The App does not knowingly collect any personal information from children under the age of 13.
